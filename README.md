# ❤️ 心电去噪探索实验平台 — ECG Denoising Lab

> **从串口采集 STM32 心电信号 → 8 种算法去噪 → 自动选最优 → 浏览器实时波形显示**
>
> 一个完整的端到端心电信号处理实验平台，集成探索分析（kai4_exploration）与实时监测（realtime_denoise）两大模块。

---

## ✨ 功能特色

### 🧪 探索分析（离线）
- **33 种去噪方法系统性对比**（FFT门控、陷波、子带/小波、SSA、PCA、ICA、自适应滤波、集成...）
- **10 张专业分析图表**：原始信号概览、噪声语谱图、PSD对比、HRV分析、平均心跳模板、雷达图
- **8 维无参考评价指标**：Peak SNR、自相关@RR、平滑度、ECG带内功率比、R峰一致性...
- **自动复合评分排序**：z-score 标准化选出综合最优方法

### 📡 实时监测（在线）
- **串口实时采集**：适配 STM32F1 13 字节二进制协议（0xAA 0xAA 帧头、CRC校验）
- **8 种算法后台批处理**：每 10 秒窗口全面分析，自动选优
- **流式滚动播放**：去噪结果以 ~62 点/块的速率平滑推送
- **R 峰实时检测**：scipy.signal.find_peaks + 自适应百分位阈值

### 🖥️ Web 可视化
- **双通道波形**：原始信号（灰色）+ 去噪信号（绿色），同屏对比
- **鼠标滚轮缩放**：普通滚轮 → 垂直（幅值）缩放 / Shift+滚轮 → 水平（时基）缩放
- **双击重置**：一键回到自动缩放
- **R 峰标记**：红色圆点标注检测到的心跳位置
- **实时指标面板**：心率、Peak SNR、信号强度、当前最优方法

## 🚀 快速开始

### 环境要求

- Python 3.8+
- Windows（串口通信依赖 Win32 API / pyserial）

### 安装依赖

```bash
pip install flask numpy scipy scikit-learn pyserial
```

### 模拟模式运行（无需硬件）

```bash
cd realtime_denoise
python main.py
```

浏览器打开 **http://localhost:8080**

等待 10 秒（数据积累阶段），即可看到去噪波形滚动播放。

### 串口模式运行（接 STM32 硬件）

```bash
python main.py --port COM37 --baud 115200
```

## 🧮 去噪算法详解

| 算法 | 类别 | 原理 | Peak SNR |
|:----|:----|:-----|:--------:|
| **FFT Gate 0.5-40Hz** 🏆 | 频域 | FFT → 平滑门控掩码 → IFFT | **+11.8 dB** |
| **Notch 50/100/150Hz + BP** | 经典 | IIR 陷波级联 ×5 + Butterworth 带通 | +9.6 dB |
| **Subband (小波类似)** | 时频 | 滤波器组多级分解 + 软阈值 | +6.9 dB |
| **SSA (奇异谱分析)** | 矩阵 | Hankel 嵌入 → SVD 截断 → 对角线平均 | +3.5 dB |
| **PCA 时延嵌入** | 统计 | 延迟坐标嵌入 → 主成分截断 → 重构 | +3.5 dB |
| **ICA + ZCR 选择** | 盲源分离 | FastICA → 过零率排序 → 优质分量重构 | +7.5 dB |
| **LMS 自适应** | 自适应 | 最小均方误差预测 → 误差滤波 | +1.7 dB |
| **等权集成** | 集成 | 前 5 种方法平均 | +2.2 dB |

> 以上 Peak SNR 来自 kai4.csv 实测数据（250Hz, 20s, 强工频干扰+基线漂移+宽带噪声）

## 📊 评估指标

| 指标 | 说明 | 最优方向 |
|:----|:-----|:-------:|
| **Peak SNR** | R峰功率 / 非峰噪声功率 | ↑ 越高越好 |
| **Autocorrelation @ RR** | R-R间期滞后处的自相关值 | ↑ 越接近1越好 |
| **Smoothness** | 二阶差分能量均值 | ↓ 越低越好 |
| **ECG Bandpower Ratio** | 0.5-40Hz 功率 / 总功率 | ↑ 越接近1越好 |
| **Peak Consistency** | R峰检出一致性 | ↑ 越接近1越好 |
